#### 내 코드 풀이 👩🏻‍💻

##### 1. 공백을 두고 두개의 수를 입력받아준다.
```readLine()```

##### 2. Split을 이용하여 공백을 기준으로 문자열을 쪼개 배열에 담아 반환한다.
```.split(separator: " ") // [1], [2]``

##### 3. map을 이용하여 배열의 요소들을 Int로 형변환 해준다.
```.map { Int($0)! }```

##### 4. reduce를 사용하여 배열의 요소들을 모두 더해준다.
```.reduce(0, +))!)```

##### 내 전체 코드!
```swift
print((readLine()?.split(separator: " ").map { Int($0)! }.reduce(0, +))!)
```
